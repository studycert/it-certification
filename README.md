# IT Certification Study Platform

A web-based platform for studying and preparing for IT certifications.

## Structure
- `index.html` - Main application
- `css/` - Stylesheets
- `js/` - JavaScript files
- `materiais/` - Study materials
- `simulados/` - Practice exams

## Getting Started
Open `index.html` in a web browser.
